# Finaura

Finaura is a clean and minimal personal finance tracking app built using **Next.js (App Router)**.  
It allows users to track expenses, view monthly analytics, and manage basic financial goals through a simple and responsive interface.

## Features

- Email/password authentication  
- Add, edit, and delete expenses  
- Monthly overview with Bar + Donut charts  
- Category-wise spending visualization  
- Auto-generated spending insights  
- Profile settings (name, salary, goal amount/year)  
- Responsive UI with clean layout  
- Custom toast notifications  
- Simple progress bar for savings vs goal

## Tech Stack

- **Next.js** (App Router)
- **React**
- **TailwindCSS**
- **Recharts**
- **JWT-based authentication**
